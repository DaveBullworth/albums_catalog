# Albums Catalog
A system for detailed cataloging of information about music albums, using Spotify.

## Technology Stack

![Alt Text](https://miro.medium.com/v2/resize:fit:1100/format:webp/1*ptqverAyBpdfUDhrs2g_3A.jpeg)

#### PERN Stack:
- ***PostgreSQL***
- ***Express***
- ***React***
- ***Node.js***

### Backend

- **Programming Language:** Node.js
- **Database:** PostgreSQL
- **Framework:** Express.js
- **ORM:** Sequelize
- **CORS Support:** cors
- **Environment Variables:** dotenv

### Frontend

- **Library:** React.js
- **State Management:** Redux Toolkit, React Redux, Redux Thunk
- **HTTP Client:** axios
- **Component Library:** Bootstrap
- **Routing:** react-router-dom
- **Styling:** Sass
- **Color Operation:** tinycolor2, colorthief
- **Animation:** mojs

## Installation and Running

### First run 

1. **Clone the repository:**
    ```bash
    git clone <URL>
    ```

2. **Install dependencies:**
    In the `client` directory:
    ```bash
    cd client
    npm install
    ```

    In the `server` directory:
    ```bash
    cd server
    npm install
    ```

### After installation

3. **Running the application:**
    In the `server` directory:
    ```bash
    npm run dev
    ```
    
    In the `client` directory:
    ```bash
    npm start
    ```
